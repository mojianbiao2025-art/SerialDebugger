## 编译说明

### 编译脚本
- `build_simple.bat` - 简单编译脚本（推荐）
- `build_qt.bat` - Qt 编译脚本
- `clean.bat` - 清理编译文件
- `rebuild.bat` - 重新编译

### 编译要求
- Qt 5.6.3
- MSVC 2013 (64-bit)
- CMake 3.5+

### 编译步骤
```bash
# 1. 设置 Qt 路径（已在脚本中配置）
# 2. 运行编译脚本
build_simple.bat

# 3. 输出文件位置
build\bin\SerialDebugger.exe
```

## 使用说明

### 基本使用
1. 启动程序（默认中文界面）
2. 选择串口和参数
3. 点击"打开串口"
4. 在发送区输入数据
5. 点击"发送"按钮

### 切换语言
1. 点击菜单栏"语言"
2. 选择需要的语言
3. 界面立即切换

### 数据绘图
1. 切换到"波形图"标签页
2. 发送格式：`plotter 1.23 4.56 7.89`
3. 或直接发送数字：`1.23 4.56 7.89`
4. 最多支持6个通道

### 修改翻译
1. 用文本编辑器打开 `lang_*.ini` 文件
2. 修改等号右边的文本
3. 保存文件
4. 重启程序
